# Kairo Scholar

## AI-Powered Study Assistant — Free Study Mode

Kairo Scholar is a modern study platform designed to help students turn notes into structured learning material. Users can upload notes or paste text to generate organized study resources such as summaries, flashcards, quizzes, and study guides.

This launch version runs in **Free Study Mode**, meaning the platform does not require paid AI APIs or external AI billing systems. The backend AI route is disabled by default to prevent unexpected API costs and keep the platform accessible for students.

---

## Features

* Upload or paste study notes
* Generate structured study guides
* Create summaries from notes
* Build flashcards for revision
* Generate practice quiz questions
* Extract key concepts and important terms
* Mobile-friendly responsive design
* Download study materials
* Clean futuristic interface
* Safe public launch with no API billing risk

---

## Free Study Mode

Kairo Scholar currently runs using a local fallback system only.

### What this means

* No `OPENAI_API_KEY` required
* No external AI billing
* Backend AI calls are disabled by default
* No risk of unexpected API charges
* Students can access free study tools safely

This allows the platform to launch publicly without generating API costs.

---

## Vision

Kairo Scholar was created to help students study smarter by transforming raw notes into organized learning material that is easier to review and understand.

The long-term vision is to evolve Kairo Scholar into a complete AI learning assistant with advanced personalized learning tools, adaptive quizzes, and mobile app support.

---

## How It Works

1. Upload notes or paste study material
2. The system processes the content
3. Kairo Scholar generates:

   * study guides
   * summaries
   * flashcards
   * quizzes
   * key concepts
4. Download or review the generated material
